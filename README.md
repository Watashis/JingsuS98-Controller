# PyS98 (Jingsu S98 Python Library)

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Python](https://img.shields.io/badge/python-3.6+-blue.svg)

Библиотека на Python для управления механической клавиатурой **Jingsu S98** (или совместимыми) по USB (HID). Позволяет управлять подсветкой, проверять уровень заряда батареи и настраивать индивидуальные цвета клавиш (Per-Key RGB).

## Особенности (Features)
- 🔋 **Мониторинг батареи**: получение текущего процента заряда аккумулятора.
- 🌈 **Глобальная подсветка**: установка единого статичного цвета для всей клавиатуры.
- 🎨 **Встроенные эффекты**: переключение режимов анимации (Breath, Spectrum, Rotating и др.) с настройкой скорости, яркости и базового цвета.
- ⌨️ **Per-Key RGB**: индивидуальная настройка цвета для каждой отдельной клавиши.

## Требования (Requirements)
- Python 3.6+
- Библиотека `hid` (`pip install hid`)

## Установка (Installation)
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/pys98.git
   cd pys98
   ```
2. Установите зависимости:
   ```bash
   pip install hid
   ```
3. *(Для Windows)* Может потребоваться `hidapi.dll` в системных путях, хотя обычно пакет `hid` ставится со всем необходимым.

## Использование (Usage)

### Инициализация
```python
from keyboard_backlight_library import KeyboardBacklight

# Vendor ID и Product ID могут отличаться, проверьте их в диспетчере устройств
VID = 0x05AC
PID = 0x024F

kb = KeyboardBacklight(vid=VID, pid=PID)
```

### Узнать уровень заряда батареи
```python
battery = kb.get_battery_level()
print(f"Заряд батареи: {battery}%")
```

### Включение встроенных режимов подсветки
Доступные режимы: `Static`, `Breath`, `Spectrum`, `Rotating`, `Explode`, `Flowing` и другие.

```python
# Включаем пульсацию красного цвета (максимальная яркость, средняя скорость)
kb.set_builtin_mode('Breath', r=255, g=0, b=0, brightness=5, speed=3)

# Выключить подсветку
kb.set_builtin_mode('Off')
```

### Настройка глобального цвета
```python
kb.set_global_color(r=0, g=255, b=0) # Вся клавиатура станет зеленой
```

### Настройка каждой клавиши (Per-Key RGB)
> [!WARNING]
> При использовании индивидуальной настройки клавиш, данные записываются в EEPROM (flash-память) клавиатуры. **Не вызывайте этот метод для создания собственных анимаций (чаще 1 раза в несколько секунд)**, так как это может привести к износу памяти и зависанию контроллера.

```python
# 1. Заливаем всю клавиатуру тусклым синим цветом
layout = {key: (0, 0, 30) for key in kb.VALID_KEYS}

# 2. Красим WASD в красный
for key in ['W', 'A', 'S', 'D']:
    layout[key] = (255, 0, 0)

# 3. Применяем конфигурацию
kb.set_per_key_colors(layout)
```

## Как найти VID и PID
Если библиотека не находит вашу клавиатуру, возможно, у нее другие `VID` и `PID`.
- **Windows**: Диспетчер устройств -> Клавиатуры -> Свойства -> Сведения -> ИД оборудования (Ищите `VID_XXXX&PID_YYYY`).
- **Linux**: Команда `lsusb`.

## Завершение работы
Не забывайте закрывать соединение после завершения отправки команд:
```python
kb.close()
```
